# GitHub Setup Instructions

## Create GitHub Repo

```bash
# Via GitHub CLI (si installé)
gh repo create benjaminbelaga/pricing-engine --public --description "Moteur de pricing unifié pour transporteurs internationaux (La Poste, Spring, FedEx, UPS)"

# Ou via web:
# https://github.com/new
# Repo name: pricing-engine
# Description: Moteur de pricing unifié pour transporteurs internationaux
# Public
# NO README, NO .gitignore (déjà créés)
```

## Push to GitHub

```bash
cd ~/repos/pricing-engine

# Add remote
git remote add origin git@github.com:benjaminbelaga/pricing-engine.git

# Push
git push -u origin main

# Verify
git remote -v
```
